# Miranda notebooks

## An interactive note-taking app built with React + TypeScript + Vite.

Miranda Notebooks lets you create dynamic, notebook-style documents composed of individual notes (cells). This UI is inspired by Jupyter Notebooks - the interactive computing platform.
